## Start Here

```bash
curl -fsSL https://skillx.run/install.sh | sh
```

```bash
skillx run github:skillx-run/skillx/examples/skills/name-poem "Your Name"
```

skillx gives you a zero-install way to fetch, scan, inject, run, and clean up Agent Skills in one command. If you only want one successful run to prove everything works, start with the GitHub example above.
